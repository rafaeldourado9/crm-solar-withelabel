# ✅ CHECKLIST - INFORMAÇÕES NECESSÁRIAS

## 🔐 AWS (obrigatório)

### 1. Credenciais IAM
```
AWS_ACCESS_KEY_ID=AKIA...
AWS_SECRET_ACCESS_KEY=...
AWS_REGION=us-east-1
```

### 2. Chave SSH
**Opção A:** Nome de chave existente
```
SSH_KEY_NAME=minha-chave
```

**Opção B:** Criar nova (me diga apenas)
```
"Criar nova chave SSH"
```

### 3. Senha RDS
```
DB_PASSWORD=SenhaForte@123
```
(mínimo 8 caracteres, letras e números)

## 🐙 GitHub (obrigatório)

### 1. Repositório
